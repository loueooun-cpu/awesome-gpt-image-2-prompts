## ❓ ¿Qué es GPT Image 2?

**GPT Image 2** (también conocido como GPT-Image-2 o gpt-image-2) es el modelo más reciente de OpenAI para generación y edición de imágenes, integrado nativamente en ChatGPT y disponible a través de la API de OpenAI.

**Capacidades clave:**
- **Generación de texto a imagen** — Crea imágenes fotorrealistas, ilustraciones, pósters, mockups de UI y más a partir de prompts en lenguaje natural
- **Edición de imágenes** — Modifica imágenes existentes con instrucciones de texto (inpainting, outpainting, transferencia de estilo)
- **Conversaciones multi-turno** — Refina imágenes iterativamente a través del diálogo
- **Renderizado de texto de alta fidelidad** — Renderiza texto con precisión dentro de las imágenes generadas
- **Generación consistente de personajes** — Mantiene la identidad del personaje en múltiples generaciones

**Por qué los desarrolladores usan la API de GPT Image 2:**
- Una sola llamada API para generación y edición
- Adherencia superior al prompt comparado con modelos anteriores
- Soporte nativo para relaciones de aspecto, transparencia y generación por lotes
- Funciona con el formato estándar de la API de OpenAI (`/v1/images/generations`)

> Aprende más sobre el uso de la API en la sección [Use GPT Image 2 API](#-use-gpt-image-2-api) a continuación.

## 🔌 Use GPT Image 2 API

Want to move from prompt inspiration to real image generation fast? Use the GPT Image 2 API docs together with the callable skill.

### 1. Quick Install the Skill

- [Go to gpt-image-2-gen-skill repository](https://github.com/EvoLinkAI/gpt-image-2-gen-skill)
- One-line install:

```bash
npx evolink-gpt-image -y
```

### 2. Quick Connect to the API

- Connect command:

```bash
curl --request POST \
  --url https://api.evolink.ai/v1/images/generations \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
  "model": "gpt-image-2",
  "prompt": "A beautiful colorful sunset over the ocean"
}'
```

- [Get your API key](https://evolink.ai/dashboard?utm_source=github&utm_medium=readme&utm_campaign=awesome-gpt-image-2-API-and-Prompts)
